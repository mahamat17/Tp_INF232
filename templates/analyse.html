📐 Formules et résultats détaillés

Moyenne (μ)

μ = (Σ xᵢ) / n

Variance (σ²)

σ² = Σ (xᵢ - μ)² / n

Écart-type (σ)

σ = √σ²

Covariance (Cov)

Cov(X,Y) = Σ (xᵢ - μx)(yᵢ - μy) / n

Coefficient de corrélation (r)

r = Cov(X,Y) / (σx · σy)

Coefficient de détermination (R²)

R² = r²

Droite de régression

y = ax + b avec a = Cov(X,Y) / Var(X) et b = μy - a·μx
